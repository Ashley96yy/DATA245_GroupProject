# DATA245 Group Project

This repository contains the code for our DATA245 group project on NYC building sustainability using the NYC Building Energy and Water Disclosure dataset.

## Team Code Structure

### Shared team files
- `Shared_EDA.ipynb`
- `Shared_EDA.py`

These files handle shared dataset inspection, common cleaning rules, and export of the shared-cleaned dataset used by all team members.

### Individual files
- `Ying_Zhu_EDA_Model.ipynb` / `Ying_Zhu_EDA_Model.py`  
  ENERGY STAR Score prediction

- `Yuyao_Ding_EDA_Model.ipynb` / `Yuyao_Ding_EDA_Model.py`  
  Weather-Normalized Site EUI prediction

- `Chao_Zheng_EDA_Model.ipynb` / `Chao_Zheng_EDA_Model.py`  
  Building performance clustering

- `Jim_He_EDA_Model.ipynb` / `Jim_He_EDA_Model.py`  
  GHG emissions intensity prediction
